Описание приложения

Программа предназначена для классификации месторождений по действующим нормативным документам. Сбор основной геолого-промысловой информации по месторождению (сведения о недропользователе, геологические данные, данные по запасам УВС с указанием величин и их категорийности по принятым документам), учет наличия закачки на месторождении/залежи, данные по геологической и гидродинамической моделям, а также данные по действующим системам сбора и подготовки продукции скважин. Сохранение данных в формате MS Excel, с возможностью добавления дополнительного листа с информацией (в случае многопластового месторождения). Дополнительная возможность - создание шаблона файла протокола ЦКР в зависимости от классификации месторождения с характеристикой УВ-сырья и основных технологических показателей.

Системные требования

Для стабильной работы приложения необходимы следующие системные требования:

Минимальные системные требования:
  • операционная система Windows XP (32-bit);
  • установленный Microsoft Office 2003;
  • установленный Microsoft .NET Framework 3.5;
  • процессор с тактовой частотой 1,2 ГГц;
  • объем свободного места на жестком диске - не менее 50 МБ;
  • объем оперативной памяти - 2,0 ГБ.
Рекомендуемые системные требования:
  • операционная система Windows 7 (64-bit) или выше;
  • установленный Microsoft Office 2010 или выше;
  • установленный Microsoft .NET Framework 4.0 или выше;
  • процессор с тактовой частотой 2 ГГц;
  • объем свободного места на жестком диске - не менее 50 МБ;
  • объем оперативной памяти - 4,0 ГБ.
